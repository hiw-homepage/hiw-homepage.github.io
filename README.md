# hiw-homepage.github.io
## 注意
  - 実際のドメインはリポジトリ名とは異なります
  - ファイルの更新メッセージは内容とは関係なく同一で行われています
  - リミックスや構築に使用する際は***クエリを流用しない***でください
## 機能するファイルとフォルダー
  ### image - フォルダー
  画像が格納されています
  ### content.html - HTMLファイル
  ライセンスが記述されています
  ### favicon.ico - アイコンファイル
  ページアイコンです
  ### index.css - CSSファイル
  ほぼすべてのページで使用されるスタイルを設定しています
  ### index.html HTMLファイル
  ドメインにアクセスしたとき最初に表示されるページです
  ### join.html HTMLファイル
  入会に関する規約とフォームを表示します
  ### mail.html HTMLファイル
  メールフォームを送信するときに使用されます
